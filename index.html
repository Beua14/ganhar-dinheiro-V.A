📜 NORMA DE GANHOS
Vazados Angola

1. Objetivo

Esta norma define as regras de ganhos dentro do aplicativo Vazados Angola, através do sistema de convites e instalação de novos usuários.

2. Como Funciona o Ganho

  • Cada usuário pode convidar amigos para instalar o aplicativo.
  • O convite deve resultar em instalação real do app.
  • O sistema contabiliza apenas instalações válidas.

3. Recompensa

A cada 10 amigos que instalarem o aplicativo, o usuário recebe:

➡️ 700 Kz (setecentos kwanzas) O valor é acumulativo — 20 convites = 1.400 Kz

4. Condições para Validar Ganho

  • O amigo convidado deve instalar o app através do link do usuário.
  • O app deve ser aberto pelo novo usuário pelo menos uma vez.
  • Não são permitidas instalações falsas ou repetidas.

5. Proibições

  • Criar contas falsas para ganhar dinheiro
  • Usar métodos automáticos ou ilegais
  • Tentar manipular o sistema

👉 Contas suspeitas terão ganhos anulados.

6. Pagamento

  • Crédito interno no app ou método definido pela plataforma
  • O pagamento será feito após validação das instalações

7. Atualizações

O Vazados Angola pode alterar valores, atualizar regras ou suspender o programa a qualquer momento.

8. Aceitação

Ao participar, o usuário aceita automaticamente todos os termos desta norma.
